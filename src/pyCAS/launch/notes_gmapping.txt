Commands to map area:
# create model in gazebo
>> roslaunch gazebo_ros empty_world.launch

# launch gazebo and model - takes in model as arguments
>> roslaunch pyCAS gmapping.launch

# launch the gmapping algo and rviz
>> roslaunch turtlebot3_slam turtlebot3_slam.launch slam_methods:=gmapping

# autonomously drive 
>> roslaunch turtlebot3_gazebo turtlebot3_simulation.launch
